# 股票预测系统使用说明

## 概述

本项目提供了两个股票预测程序：

1. **interactive_stock_prediction.py** - 基于Kronos AI模型的完整预测系统
2. **simple_stock_prediction.py** - 基于技术分析的简化预测系统

## 功能特点

### 主要功能
- 支持用户输入股票代码或上传TXT文件
- 自动下载股票K线数据（使用Akshare）
- 基于历史100个交易日数据预测未来一个月走势
- 生成预测结果图表和数据文件
- 支持批量股票预测

### 数据源
- 使用Akshare获取A股历史数据
- 支持前复权数据
- 自动处理交易日历（排除周末）

## 安装依赖

```bash
pip install pandas matplotlib numpy akshare torch transformers
```

## 使用方法

### 1. 基于Kronos AI模型的预测（推荐）

```bash
python interactive_stock_prediction.py
```

**特点：**
- 使用先进的Kronos AI模型
- 预测精度更高
- 需要GPU支持（推荐）
- 模型加载时间较长

### 2. 基于技术分析的简化预测

```bash
python simple_stock_prediction.py
```

**特点：**
- 基于技术指标分析
- 无需GPU，CPU即可运行
- 启动速度快
- 预测结果仅供参考

## 使用步骤

### 1. 启动程序
运行相应的Python文件

### 2. 选择输入方式
- **手动输入**：直接输入股票代码，多个代码用逗号分隔
- **文件输入**：从TXT文件读取股票代码列表

### 3. 股票代码格式
- 支持6位数字A股代码
- 示例：600030, 002261, 688326, 300364

### 4. 查看结果
- 程序会自动生成预测图表
- 保存预测结果到CSV文件
- 显示预测数据预览

## 文件说明

### 输入文件格式（stock_codes.txt）
```
# 这是注释行
600030
002261
688326
300364
```

### 输出文件
- `prediction_股票代码_时间戳.png` - 预测图表
- `prediction_results_股票代码_时间戳.csv` - 预测数据

## 预测参数

### 历史数据
- 使用最近100个交易日的数据
- 自动排除周末和节假日

### 预测期间
- 预测未来30个交易日
- 自动排除周末

### 数据字段
- 开盘价 (open)
- 最高价 (high)
- 最低价 (low)
- 收盘价 (close)
- 成交量 (volume)
- 成交额 (amount)
